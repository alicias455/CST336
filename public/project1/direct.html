The direct action virus is another type of infector virus. Different from the resident, this virus is
seen as "non-resident" because it does not install or hide itself on the computer's memory. As
mentioned on the main page, it actually attaches itself to a specific file type. In addition, the
direct action virus does not usually remove any files, although making some files inaccessible, or
obstruct the system's performance. This virus may be much easier to remove with an anti-virus
software, unlike the resident file that sometimes requires a unique tool. Rugrat, a direct action
virus, has been rumored to be the first infection written in the Intel Itanium instruction set. Also
known as Win64.Rugrat, it was only able to infect Itanium executable files.
